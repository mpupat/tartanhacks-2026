# WinBack
### Team Bonjour Bombay submission for Tartanhacks 2026.

Winback is a smart financial solution that helps you invest your credit card cashback to win back extra money. Winback is backed by Kalshi to provide smart investing options and XRP Ledger blockchain to log all transactions for security and transparency. At the end of each month, you win all the money linked to your correct predictions. And since you are only reinvesting your cashback, in case of prediction errors, you donâ€™t lose any money!

# Installation
You need Python and node.js to run this project
It has been tested with Python 3.13 and node.js 25.2.1

## Run the backend first
create a python virtual environment :
```
python -m venv .venv
source .venv/bin/activate
cd backend
pip install -r requirements.txt
python main.py
```
## Run the frontend then:
```
cd frontend
npm install
npm run dev
```
